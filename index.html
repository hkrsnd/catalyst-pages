What do the metrics mean?
Success Rate
How many optimization tasks completed with an improved molecule vs. total tasks attempted.
LogP Change
Average change in lipophilicity (fat-solubility). Negative = more water-soluble; positive = more fat-soluble. Drug-like range: −0.4 to 5.6.
Total Tasks
Number of separate molecule-optimization tasks run in the experiment. Each task starts from a different molecule.
Runtime
Total wall-clock time for all tasks in the experiment to complete, including agent discussion and property evaluation.
